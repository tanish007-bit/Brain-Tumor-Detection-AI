# Brain-Tumor-Detection-AI
This project detects brain tumors from MRI images using a CNN model.

## Technologies Used
- Python
- TensorFlow
- Keras
- OpenCV
- Flask

## Dataset
Br35H :: Brain Tumor Detection 2020

## How it Works
1. Upload MRI image
2. Image preprocessing
3. CNN model prediction
4. Result shown on web page

## Run Project

Install libraries

pip install -r requirements.txt

Run the application

python app.py

Open browser

http://127.0.0.1:5000
